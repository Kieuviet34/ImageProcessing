# ImageProcessing
Project xử lý ảnh sử dụng OpenCV

# Yêu cầu
Tạo môi trường ảo và kích hoạt môi trường (window, linux, mac)
```sh
    python3 -m venv .venv
```
kích hoạt (window):
```sh
    .venv/Scripts/activate
```
Trên linux, macOS:
```bash
    source .venv/bin/activate
```


Cài đặt các thư viện trong requirement.txt
```sh
    pip install -r requirement.txt
```
hoặc 
```bash
    pip3 install -r requirement.txt
```
# Tải datasets
Chạy file getData.py để tải MNIST dataset từ kagglehub

```bash
    python3 getData.py
```
